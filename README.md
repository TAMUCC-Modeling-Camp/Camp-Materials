# Summer Modeling Camp Publish Files

This folder contains the publish-ready PDF materials for the Mathematical Modeling & Computational Projects Camp.

## Contents

- `day1-lecture.pdf` through `day8-lecture.pdf`
- `day1-exercises.pdf` through `day8-exercises.pdf`

## Purpose

These files are the final lecture notes and exercise handouts prepared for printing or sharing. The source LaTeX files live elsewhere in the project; this folder is intended only for the compiled PDFs.

## Use

Open or distribute the PDF files directly. If any source files are updated, recompile them and replace the PDFs in this folder.

## Notes

- All files here are PDF outputs only.
- This folder is intended to stay clean and publication-ready.
